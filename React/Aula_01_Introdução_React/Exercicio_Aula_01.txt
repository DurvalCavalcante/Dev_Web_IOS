React :
É uma biblioteca para construir interface, usada mais pelos Desenvolvedores Fornt-End. Para aplicação de SPA ( Single Page Applicantion ) e MPA ( Multi-Page App ). O SPA é voltado mais para o lado do cliente, e ela faz as transições entre páginas mais rápidas. E o MPA  é um aplicação com várias páginas, que consiste em um grande número de páginas que atualiza toda vez que os dados são alterados.
